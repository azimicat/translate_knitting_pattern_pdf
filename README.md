# Knitting Translator

英語の棒針・かぎ針編みパターン PDF を日本語に翻訳して、バイリンガル PDF として出力する macOS アプリです。

Google AI (Gemini 2.5 Flash) を使い、**無料枠のみで動作します（月額料金不要）。**

---

## 動作環境

- macOS 14 Sonoma 以降
- Google AI API キー（無料）

---

## セットアップ

### 1. アプリをダウンロードする

[Releases](../../releases) ページから最新の `KnittingTranslator.zip` をダウンロードして展開し、アプリを起動します。

### 2. Google AI API キーを取得する

1. [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセス
2. **「Create API key」** をクリックしてキーを生成
3. アプリ起動時に表示される入力シートにキーを貼り付けて「保存」

> 無料枠：1,500 リクエスト/日（1 ページ = 1 リクエスト）

---

## 使い方

1. **翻訳モード** を選択（棒針 / かぎ針）
2. 英語の編みパターン PDF をドロップゾーンにドラッグ、または「ファイルを選択」で指定
3. **「翻訳を開始」** をクリック
4. 完了後、プレビューで確認して **「PDF を保存...」** で出力

進捗バーがページ単位で進みます（0〜90%: 翻訳 / 90〜100%: PDF 生成）。
翻訳中に **「キャンセル」** をクリックすると処理を中断できます。
API キーを変更したい場合は画面右端の **🔑 ボタン** から変更できます。

---

## 注意事項

- スキャン画像のみの PDF（テキストレイヤーなし）は翻訳できません
- 1 回の翻訳でページ数分の API リクエストが発生します。無料枠（1,500 リクエスト/日）を超えそうな場合はアプリ内でアラートが表示されます
