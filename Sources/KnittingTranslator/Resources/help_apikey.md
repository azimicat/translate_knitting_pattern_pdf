# Google AI APIキーの取得方法

このアプリを使うには、Googleが提供する「APIキー」が必要です。
**完全無料**で取得でき、クレジットカードの登録も不要です。

所要時間の目安：**5〜10分**

---

## APIキーとは？

APIキーとは、アプリがGoogleの翻訳サービスを使うための「合言葉」のようなものです。
このキーをアプリに登録することで、アプリがあなたの代わりにGoogleに翻訳を依頼できるようになります。

キーは英数字の長い文字列（例：`AIzaSyAbc123...`）です。

---

## 手順

### ステップ 1：Googleアカウントを用意する

Google AI Studioを使うにはGoogleアカウント（Gmailなど）が必要です。

- すでにGmailをお持ちの方はそのアカウントで使えます
- お持ちでない方は `accounts.google.com/signup` から無料で作成できます

### ステップ 2：Google AI Studioを開く

以下のURLをブラウザのアドレスバーに入力して開きます。

`aistudio.google.com/app/apikey`

ログイン画面が表示されたらGoogleアカウントでログインしてください。

### ステップ 3：APIキーを作成する

1. 青いボタン **「Create API key」**（または「APIキーを作成」）をクリックします
2. 「Create API key in new project」をクリックします
3. しばらく待つと `AIzaSy` から始まる長い文字列が表示されます。これがAPIキーです。

### ステップ 4：APIキーをコピーする

表示されたAPIキーの右側にある **コピーアイコン** をクリックします。

> **大切なこと：** APIキーは他の人に教えないようにしてください。キーを知っている人があなたの無料枠を使い切ってしまう場合があります。

### ステップ 5：アプリにAPIキーを登録する

1. 入力欄にコピーしたAPIキーを貼り付けます（**Command + V**）
2. **「保存」** ボタンをクリックします

2回目以降はキーが自動保存されるので、毎回入力する必要はありません。

---

## 無料枠について

- **1日のリクエスト数：** 500回まで
- **1分あたりのリクエスト数：** 15回まで
- **リセットのタイミング：** 毎日深夜0時（日本時間）

1リクエスト＝1ページの翻訳に相当します。たとえば10ページのPDFを翻訳すると10回消費されます。上限に近づくとアプリが警告を表示します。

---

## よくある質問

**Q. クレジットカードの登録は必要ですか？**
無料枠を使う限り不要です。カード情報の入力を求められたら、有料プランへの移行画面です。そのまま閉じて問題ありません。

**Q. 「利用規約に同意する」という画面が出ました**
Googleの利用規約です。内容を確認の上、同意してから進んでください。

**Q. ページが英語で表示されました**
操作は同じです。「Create API key」ボタンをクリックしてください。ブラウザの翻訳機能（右クリック→「日本語に翻訳」）を使うと読みやすくなります。

**Q. APIキーをなくしてしまいました**
Google AI Studioに再度アクセスすると、作成済みのキーの一覧が表示されます。コピーアイコンから再取得できます。一覧にない場合は「Create API key」で新しいキーを作り直してください。

---

## うまくいかないときは

- ブラウザを再読み込み（**Command + R**）してからやり直す
- ブラウザを変えてみる（SafariがうまくいかないときはChromeを試す）
- 一度ログアウトしてから再度ログインする
