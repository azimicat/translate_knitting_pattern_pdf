# Google AI API キーの取得方法

このアプリを使うには、Google が提供する「API キー」というものが必要です。
**完全無料**で取得でき、クレジットカードの登録も不要です。

所要時間の目安：**5〜10分**

---

## API キーとは？

API キーとは、アプリが Google の翻訳サービスを使うための「合言葉」のようなものです。
このキーをアプリに登録することで、アプリがあなたの代わりに Google に翻訳を依頼できるようになります。

キーは英数字の長い文字列（例：`AIzaSyAbc123...`）です。

---

## 手順

### ステップ 1：Google アカウントを用意する

Google AI Studio を使うには Google アカウント（Gmail など）が必要です。

- すでに Gmail をお持ちの方はそのアカウントで使えます
- お持ちでない方は [https://accounts.google.com/signup](https://accounts.google.com/signup) から無料で作成できます

---

### ステップ 2：Google AI Studio を開く

以下の URL をブラウザ（Safari、Chrome など）のアドレスバーにコピー＆ペーストして開きます。

```
https://aistudio.google.com/app/apikey
```

**ログイン画面が表示されたら：**

1. 「次へ」をクリック
2. Google アカウントのメールアドレスを入力 → 「次へ」
3. パスワードを入力 → 「次へ」

---

### ステップ 3：API キーを作成する

ページが開いたら、次のような画面になります。

![API キー管理ページのイメージ]

1. 青いボタン **「Create API key」**（または「APIキーを作成」）をクリックします

2. 「Search for or select a project below」という画面が出ることがあります。
   その場合は **「Create API key in new project」** をクリックしてください。
   （既存のプロジェクトを選ぶ必要はありません）

3. しばらく待つと、`AIzaSy` から始まる長い文字列が表示されます。
   これが API キーです。

---

### ステップ 4：API キーをコピーする

1. 表示された API キーの右側にある **コピーアイコン（重なった四角のマーク）** をクリックします
2. 「コピーしました」などのメッセージが出ればコピー完了です

> **大切なこと：** API キーは他の人に教えないようにしてください。
> キーを知っている人があなたの無料枠を使い切ってしまう場合があります。
> メモする場合はご自身だけが見られる場所に保管してください。

---

### ステップ 5：アプリに API キーを登録する

1. KnittingTranslator アプリを起動します
2. 起動直後に「API キーを入力してください」という画面が表示されます
3. 先ほどコピーした API キーを入力欄に貼り付けます
   （貼り付けはキーボードの **Command + V** です）
4. **「保存」** ボタンをクリックします

これで設定完了です。

> **2回目以降：** キーは自動的に保存されるので、毎回入力する必要はありません。
> キーを変更したい場合は、アプリ画面右上の **鍵のアイコン（🔑）** から変更できます。

---

## 無料枠について

Google AI API は無料で使えますが、1日に使える上限があります。

| 項目 | 無料の上限 |
|------|-----------|
| 1日のリクエスト数 | 500回 |
| 1分あたりのリクエスト数 | 15回 |

1リクエスト＝1ページの翻訳に相当します。
たとえば10ページのPDFを翻訳すると、10回消費されます。

上限に近づくと、アプリが翻訳前に警告を表示します。
上限はその日の深夜0時（日本時間）にリセットされます。

## よくある質問

**Q. クレジットカードの登録は必要ですか？**
A. 無料枠を使う限り不要です。カード情報の入力を求められたら、有料プランへの移行画面です。そのまま閉じて問題ありません。

**Q. 「利用規約に同意する」という画面が出ました**
A. Google の利用規約です。内容を確認の上、同意してから進んでください。

**Q. ページが英語で表示されました**
A. 操作は同じです。「Create API key」ボタンをクリックしてください。
   ブラウザの翻訳機能（右クリック → 「日本語に翻訳」）を使うと読みやすくなります。

**Q. API キーをなくしてしまいました**
A. 同じ手順で Google AI Studio にアクセスすると、作成済みのキーの一覧が表示されます。
   コピーアイコンから再取得できます。
   一覧に表示されないときは「Create API key」で新しいキーを作り直してください。

**Q. 「このアプリケーションは確認されていません」と表示されました**
A. Google AI Studio は Google 公式のサービスです。
   このメッセージが出た場合は「詳細」→「安全でないサイトに移動」をクリックして進んでください。

---

## うまくいかないときは

- ブラウザを再読み込み（**Command + R**）してからやり直す
- ブラウザを変えてみる（Safari がうまくいかない場合は Chrome を試す）
- 一度ログアウトしてから再度ログインする

それでも解決しない場合は、アプリの配布者にお問い合わせください。
