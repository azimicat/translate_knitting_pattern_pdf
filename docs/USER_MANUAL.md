# ユーザーマニュアル

## 概要

**Knitting Translator** は、英語の編み物・かぎ針編みパターン PDF を日本語に翻訳し、原文と訳文を並べた対訳 PDF を生成する macOS アプリです。

---

## 動作環境

- macOS 14 Sonoma 以降
- Google AI API キー（無料）

---

## 初回セットアップ

### API キーの取得と入力

1. アプリを起動すると、API キーの入力を求めるシートが表示されます
2. Google AI Studio で無料の API キーを取得してください（[取得方法](Google_AI_APIキーの取得方法.md)）
3. API キーを入力して「保存」をクリックします

API キーは Mac 内にのみ保存され、外部に送信されることはありません。

---

## 基本的な使い方

### 1. 翻訳モードを選択する

画面上部のピッカーで翻訳モードを選択します。

| モード | 対象 |
|---|---|
| **棒針** | 棒針編み（knitting）パターン |
| **かぎ針** | かぎ針編み（crochet）パターン |

モードに応じて適切な用語集が翻訳に使用されます。

### 2. PDF ファイルを読み込む

以下のいずれかの方法で PDF を読み込みます。

- **ドラッグ＆ドロップ**: PDF ファイルをドロップゾーンにドラッグする
- **ファイル選択**: ドロップゾーン内の「ファイルを選択」ボタンをクリックする

読み込まれたファイル名がドロップゾーンに表示されます。

### 3. 翻訳を開始する

「翻訳を開始」ボタンをクリックします。

- ページ数に応じて数分かかります（1 ページあたり約 10〜30 秒）
- プログレスバーで現在の進捗が確認できます
- 0〜90%: Gemini API による翻訳処理
- 90〜100%: PDF 生成処理

### 4. PDF を保存する

翻訳が完了すると、対訳 PDF のプレビューが表示されます。

「PDF を保存...」ボタンをクリックして保存先を選択します。

---

## 出力 PDF の仕様

| 項目 | 内容 |
|---|---|
| 用紙サイズ | A4 |
| レイアウト | 2 カラム（左: 原文、右: 日本語訳） |
| フォント（英語） | Helvetica Neue |
| フォント（日本語） | Hiragino Sans |
| テキストグループ | パターン行: 1 文 1 行 / 説明文: 段落単位 |
| 書式 | 太字・斜体を元 PDF から引き継ぎ |

---

## API 使用量について

Google AI の無料枠は **1 日あたり 500 リクエスト**（1 ページ = 1 リクエスト）です。

使用量が 90% を超えると警告アラートが表示されます。翻訳を継続するか中断するかを選択できます。使用量は毎日 UTC 深夜にリセットされます。

---

## トラブルシューティング

### 翻訳が始まらない

- API キーが正しく入力されているか確認してください
- ネットワーク接続を確認してください

### エラーメッセージが表示された

- API キーの有効期限や上限を確認してください
- PDF が破損していないか確認してください

### API キーを変更したい

画面右上の設定アイコン（または「API キーを変更」リンク）から変更できます。

---

## プライバシーについて

- API キーは Mac 内の UserDefaults にのみ保存されます
- PDF の内容は翻訳のため Google Gemini API に送信されます
- アプリ自体はデータを収集・送信しません
